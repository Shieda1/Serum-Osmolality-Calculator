The serum osmolality calculator is a tool that counts the patients serum osmolality.
How to calculate serum osmolality?
To calculate serum osmolality you should use the following formula:
serum osmolality = 2 * na + blood urea nitrogen / 2.8 + glucose / 18 + serum alcohol concetration / 3.7
